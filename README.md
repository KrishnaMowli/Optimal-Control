# Optimal Control

A collection of MATLAB implementations demonstrating fundamental and advanced optimal control techniques. This repository contains simulation-based projects focused on optimal controller design, dynamic programming, and the analysis of continuous-time linear systems.

---

## Repository Contents

### 01. Finite vs. Infinite Horizon LQR
Implementation and comparative analysis of finite- and infinite-horizon Linear Quadratic Regulators (LQR) using Differential and Algebraic Riccati Equations. The project includes controller synthesis, optimal state and costate trajectory analysis, and performance comparison.

### 02. Dynamic Programming
Implementation of Dynamic Programming algorithms for optimal control problems based on Bellman's Principle of Optimality. The project demonstrates backward recursion, optimal policy computation, and numerical solution techniques.

## Learning Objectives

This repository aims to:

- Develop a rigorous understanding of optimal control theory.
- Implement classical optimal control algorithms from first principles.
- Compare finite- and infinite-horizon optimal control formulations.
- Analyze optimal trajectories, control inputs, and costate dynamics.
- Apply Dynamic Programming techniques to deterministic optimal control problems.
